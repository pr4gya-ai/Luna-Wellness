# Luna-Wellness

Luna Wellness is a mental wellness app for young women, focused on menstrual cycle mood tracking and reproductive trauma support (pregnancy loss, infertility stress, post-procedure care).

## Status
🚧 In early development. This repository currently holds a placeholder site used to generate an HTTPS URL for app deployment via Natively.dev.

## Planned Features
- Cycle and mood tracking with phase-based insights
- PMDD vs. PMS screening (pending clinical review)
- Reproductive trauma support journaling and resources
- Crisis safety resources accessible from every screen
- Private, secure data storage

## Tech Stack (planned)
- Frontend: React Native (via Natively/Expo)
- Backend: Supabase (PostgreSQL, Auth, Storage)

## Disclaimer
This app is not a substitute for professional medical or mental health care. All clinical content, screening tools, and crisis resources will be reviewed before public release.
